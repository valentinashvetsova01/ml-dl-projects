# Machine Learning & Deep Learning Portfolio

A curated collection of machine learning, deep learning, explainable AI, graph learning, and computational neuroscience projects. Each project is organized in its own folder and includes a notebook, a concise project description, methodology highlights, and reported results.

## Projects

| Project | Area | Key result |
|---|---|---|
| [Neural Decoding Methods](./neural-decoding/) | Computational neuroscience | 99.7% accuracy and 1.962/2 bits with 50 neurons |
| [Stroke Risk Prediction](./stroke-risk-prediction/) | Healthcare ML | Test ROC AUC: 0.838 |
| [GCN Node Classification](./gcn-node-classification/) | Graph deep learning | Unknown-node Macro-F1: 0.647 |
| [Ad Click Prediction with LIME](./ad-click-prediction/) | Deep learning and XAI | Test F1: 0.963 |

## Running the Projects

The notebooks were developed in Google Colab and can also be run locally in Jupyter. Each project README contains its own notebook and Colab links.

Replace `YOUR_USERNAME` and `YOUR_REPOSITORY` in the Colab links after uploading the repository to GitHub.
